Сегодня убил пол-дня, считаю нужно рассказать всем. Есть моменты, когда после логина нужно возвращаться на то же место, с которого вошел на логин. Например, со страницы "новый пост". Залогинился и не попадаешь в корень а появляешься уже на странице "новый пост". В Джанге на этот случай, при срабатывании "LoginRequiredMixin" либо "@login_required", в реквесте "GET" добавляется параметр "next", содержащий адрес предыдущей страницы, которая логично должна быть следующей после логина. Так вот, во первых, чтобы сохранить этот адрес, нужно в темплейт добавить строку <input type="hidden" name="next" value="{{ next }}">, в любом месте внутри формы. Этот инпут не отображается а лишь передает информацию. Ну и функцию get_success_url дополнить следующим образом:
def get_success_url(self) -> str:
        url = self.get_redirect_url()
        if url:
            return url
        else:
            return reverse_lazy('home')
